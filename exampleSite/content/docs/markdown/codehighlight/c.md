# C code highlight

## Syntax and demo


**The markdown**
```markdown
#include <stdio.h>
int main()
{
    printf("hello world!\n");
    return 0;
}
```

**Rendered as follow**
```c
#include <stdio.h>
int main()
{
    printf("hello world!\n");
    return 0;
}
```
